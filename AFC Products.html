Add Value Machine

SAVM5000–Add Value Machine

It can process contactless smart cards for fare collection across different mass transit systems like bus rapid transit, metro, tram, ferry and train. It is Designed for stationary use, high durability and supports interactive display. It houses integrated ISO/IEC 14443 A/B Reader/ Writer, 7” Standard deOinition LED Display with capacitive touch screen, ARM based quad core processor and a speaker. It is compatible with entire MIFARE family and SONY FELICA. Modular Design which can easily be customised as per user requirement. Can be used in non-transit application like event ticketing, parking, tourist attractions and kiosk. Can be used as bus validators and Rechargers.

Ticket Validator

Ticket Reader Machine is a high performance contactless validator that offers fast and reliable ticket and payment validation across public transport system. The Ticket Reader Machine can read contactless based smart Cards & tokens and display Smartcard/Token validity, remaining balance and travel history to the patron. It is designed for on board and stationary use with high durability. It houses integrated ISO/IEC 14443 A/B Reader, 7” Standard definition LED Display. S-CUBE’s TRM is ideal for processing different contactless smart cards quickly and efficiently

Automatic Token Encoder

Automatic Token Encoder is used for dispenses contact less smart media token, encoding, processing of token and accordingly guiding the flow of token according to its validity into bin or outlet. Support ISO 14443 A&B, Mifare Ultralight and Sony FeliCa. Supports standard dimension token. Designed for Heavy duty operations, reliable and with high throughput. Used basically in Token Initialisation/Encoding, Ticket Office Machine, Excess Fare Office, Point of Sale Ter minals, Token Vending Machines and Kiosks. Modular design which can be customised as per clients need and requirements.

Ticket Recharge Terminal

Ticket Charging Terminal is a high performance smart card processing terminal for fare collection across different mass transit systems like bus rapid transit, metro, tram, ferry and train. It is designed for stationary use with high durability and supports interactive display. It brings better ergonomics, higher hardware variability and wider functionality. Used basically for validating and recharging the smart cards (Mifare family, Smart MX & Sony Felica). It accepts different payment method that includes currency notes (Cash) as well as credit/debit cards (Magnetic strip | chip based | contactless (optional)). Can be used in non-transit application like event ticketing, parking, tourist attractions and kiosk.

Automatic Feeder Machine

Automatic Token Dispenser is used for dispenses contact less smart media token. The token is delivered to a tray and cane be validated on external contactless reader/Writer by just sliding it. In build control card for ejection and counting of tokens. Presence of optical sensors enable precise counting of tokens. Designed for Heavy duty operations, reliable and with high throughput. Used basically in Ticket Office Machine, Excess Fare Office, Point of Sale Terminals, Token Vending Machines and Kiosks. Modular and flexible design which can be customised as per clients need and requirements. Supports standard dimension token (Mifare Ultralight & Felica)
