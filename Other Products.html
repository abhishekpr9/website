AFM IOTA CARD:

AFM IOTA card was developed for one of our esteemed customer to integrate with existing token vending machines. The functionality of system was maintained exactly the same as that was provided by original vendor with some enhancements regarding the hardware protection and power consumption.

Features:

  • Low power consumption.
  • Protection against reverse current.
  • All commands supported from single coin vending to empty command.
  • LEDs for display of various modes.
  • Completed SMA development project of DMRC (Delhi Metro Rail Service).



SMA controller:

SMA IOTA card is responsible for controlling SMA Mechanism. It includes accepting token and guiding it to their respective containers. It also provides RS422 communication path from UCM to CSC reader. SMA IOTA firmware understands communication protocols between UCM and SMA IOTA.

Features:

  • SMA IOTA firmware capable of understanding following commands from UCM:
    - Slot open commands for Blue token.
    - Slot open command for Red token.
    - Token to container command.
    - Un-Jam Command.
    - Initialize SMA command.
    - Can sense token presence.
    - Send success/error status to UCM.
  • Can work in maintenance mode.
